# markdown语法学习

## 1.标题

代码块如下：

```bash
# 一级标题 
## 二级标题
### 三级标题
```

显示如下：

# 一级标题 
## 二级标题
### 三级标题

## 2.列表

### 2.1无序列表

代码块如下：
```bash
* 1
+ 2
- 3
```
* 1
+ 2
- 3

### 2.2有序列表

代码块如下：
```bash
1. 第一行
1. 第二行
3. 第三行
```

显示如下：

1. 第一行
1. 第二行
3. 第三行

可以看到，默认会进行排序

## 3.区块引用

代码块如下：
```
>这是一个区块引用
>>这是一个二级嵌套引用
>>>这是一个三级嵌套引用
```
显示如下：

>这是一个区块引用
>>这是一个二级嵌套引用
>>>这是一个三级嵌套引用

## 4.链接

### 4.1 行内式

代码块如下：

```
[百度](https:www.baidu.com)经常出现的列表链接应该这样写：

+ [Django介绍与框架整合，并使用MySQL实现增删改查](https://my.oschina.net/epoch/blog/1788273)
+ [Python3的介绍、安装和命令行的认识](https://my.oschina.net/epoch/blog/1787262)
+ ...
```

显示如下：

[百度](https:www.baidu.com)
经常出现的列表链接应该这样写：<br>
+ [Django介绍与框架整合，并使用MySQL实现增删改查](https://my.oschina.net/epoch/blog/1788273)
+ [Python3的介绍、安装和命令行的认识](https://my.oschina.net/epoch/blog/1787262)
+ ...

### 4.2 参数式

代码如下：

```
这是我的blog,我想看到我的中文提示[我的博客](https://my.oschina.net/epoch/home "KevinBruce的博客")
```

显示如下：

这是我的blog,我想看到我的中文提示[我的博客](https://my.oschina.net/epoch/home "KevinBruce的博客")

## 6.代码块

